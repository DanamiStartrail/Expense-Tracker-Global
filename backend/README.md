# 💰 Expense Tracker App
Proyek Full-Stack untuk Manajemen Pengeluaran Pribadi

# 📌 Deskripsi Proyek
Aplikasi berbasis web untuk membantu pengguna mencatat, memantau, dan menganalisis pengeluaran harian. Proyek ini dibangun untuk mendalami implementasi database relasional, optimasi query agregasi, dan integrasi frontend-backend.

# 🚀 Fitur Utama
- Autentikasi Pengguna: Login dan register yang aman dengan enkripsi password.

- Manajemen Transaksi (CRUD): Tambah, lihat, ubah, dan hapus catatan pengeluaran.

- Kategorisasi Otomatis: Pengelompokan pengeluaran (Makanan, Transportasi, dsb).

- Dashboard Analitik: Visualisasi pengeluaran bulanan menggunakan grafik.

- Filter Data: Pencarian transaksi berdasarkan rentang tanggal tertentu.

# 🛠️ Tech Stack
- Frontend: [Contoh: React.js + Tailwind CSS]

- Backend: [Contoh: Node.js (Express)]

- Database: [Contoh: PostgreSQL / MySQL]

- ORM: [Contoh: Prisma / Sequelize]

- Tooling: Docker (untuk isolasi database di Parrot OS).

# 🗄️ Skema Database (ERD)
Proyek ini menggunakan relasi database sebagai berikut:

- Users (1:M) Transactions

- Categories (1:M) Transactions

(Tips: Anda bisa menyisipkan gambar ERD atau tabel Markdown di sini nantinya)
